# 🇮🇳 Indian Economy Analytics Dashboard

## 📌 Project Description

The Indian Economy Analytics Dashboard is a real-world data analytics project designed to analyze the financial health and economic stability of Indian middle-class households across different states.

This project focuses on understanding the impact of:

- Inflation
- Cost of living
- Rent burden
- Savings behavior
- Employment stability
- Financial stress

using Python-based data analysis and visualization techniques.

The dashboard was developed using:
- Pandas for data manipulation
- NumPy for numerical analysis
- Matplotlib & Seaborn for visualization
- Streamlit for interactive dashboard development

The project includes:
- KPI engineering
- Economic stability analysis
- Financial stress scoring
- State-wise comparison
- Correlation analysis
- Interactive filtering dashboard

This project simulates a real-world business intelligence and economic analytics system that can help identify financially vulnerable regions and analyze affordability challenges in India.

## 🎯 Business Problem Solved

This project helps analyze how inflation, expenses, and housing costs affect the financial stability of Indian households.

The dashboard identifies:
- states with high financial stress,
- areas with lower savings capability,
- economic vulnerability patterns,
- and affordability challenges.

---

## 📈 Insights Generated

- Metro cities have higher salaries but also higher living expenses.
- Inflation significantly reduces purchasing power.
- Rent burden directly impacts household savings.
- States with higher employment rates show better economic resilience.
- Lower-income states are more vulnerable to inflation shocks.
